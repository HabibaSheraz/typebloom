# Core Routing Structure

| Route | Purpose |
| --- | --- |
| `/` | Public landing page |
| `/dashboard` | Personalized progress and quick actions |
| `/learn` | Course catalog and learning path |
| `/lesson` | Active lesson experience |
| `/typing-test` | Timed typing tests |
| `/code-typing` | Code typing mode |
| `/games` | Typing games hub |
| `/progress` | Analytics and weak-key reports |
| `/achievements` | XP, levels, and achievement collection |
| `/leaderboard` | Opt-in rankings |
| `/profile` | User profile and privacy controls |
| `/settings` | Appearance, typing, accessibility, privacy, and notification preferences |

Future auth routes:

| Route | Purpose |
| --- | --- |
| `/auth/sign-in` | Sign in |
| `/auth/sign-up` | Create account |
| `/auth/forgot-password` | Request password reset |
| `/auth/reset-password` | Complete password reset |
| `/onboarding` | First-run wizard |
